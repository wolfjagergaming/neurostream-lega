Politique de Confidentialité

Dernière mise à jour : 15 mars 2026

Résumé : NeuroStream Bot collecte un minimum de données nécessaires à son fonctionnement. Vos clips restent sur votre ordinateur. Aucune donnée n'est vendue à des tiers.

1. Données collectées

Nous collectons uniquement les données suivantes :

Donnée Usage Stockage
ID Serveur Discord Identifier où publier les clips Local (votre PC)
ID Channels Discord Savoir où poster Local (votre PC)
Clips vidéo Contenu à publier Local uniquement
Configuration bot Préférences utilisateur Local (votre PC)

2. Ce que nous NE collectons PAS

  • ❌ Messages privés Discord
  • ❌ Données personnelles (nom, email, etc.)
  • ❌ Historique de navigation
  • ❌ Informations de paiement
  • ❌ Contenu des serveurs sans permission

3. Utilisation des données

Les données sont utilisées exclusivement pour :

  • Publier les clips sur les channels Discord configurés
  • Envoyer les notifications demandées
  • Fonctionner correctement sur votre système

4. Stockage des données

Important : Toutes vos données sont stockées localement sur votre ordinateur. Nous n'avons pas accès à vos clips ni à votre configuration.

5. Partage des données

Nous ne vendons, n'échangeons ni ne transférons vos données à des tiers. Les seuls services externes utilisés sont :

  • Discord API : pour publier sur votre serveur
  • YouTube API (optionnel) : si vous activez l'upload YouTube
  • Twitch API : pour récupérer les clips

6. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : voir vos données stockées
  • Droit de rectification : modifier votre configuration
  • Droit à l'effacement : supprimer toutes vos données en désinstallant le bot
  • Droit à la portabilité : exporter votre configuration

7. Sécurité

Nous mettons en œuvre les mesures suivantes :

  • Stockage local chiffré des tokens d'accès
  • Aucune transmission de données non nécessaire
  • Mises à jour régulières de sécurité

8. Cookies et technologies similaires

Le Bot Discord n'utilise pas de cookies. Seul le site web (si applicable) peut utiliser des cookies essentiels au fonctionnement.

9. Modifications de cette politique

Cette politique peut être mise à jour. Les changements seront notifiés via le Bot ou sur notre site web.
